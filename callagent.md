# エージェント呼び出しフレームワーク

ユーザーの依頼に応じて適切なエージェントを `subagent_type` で呼び出す。

---

## ルーティング優先順位

1. ユーザーがキャラ名・トリガーワードを明示した場合 → 対応エージェントを呼ぶ
2. タスクの性質からエージェントが自明な場合 → 自動ルーティング
3. どれにも該当しない場合 → Claude Code 本体が処理

---

## 登録エージェント

### mephi
- **subagent_type**: `mephi`
- **所属**: 内部
- **明示トリガー**: メフィ / メフィーチェック / セキュリティ検証
- **自動ルーティング**: セキュリティリスクの判断・監査・脆弱性診断が必要なとき

### alice
- **subagent_type**: `alice`
- **所属**: 内部
- **明示トリガー**: アリス / インフラ / 実装して / 作って
- **自動ルーティング**: システム開発・運用・技術的な実装が必要なとき

---

## エージェント登録手順

新しいエージェントを追加する場合:

1. `~/.claude/agents/{name}.md` にSOULファイルを配置する
2. 上記「登録エージェント」セクションにブロックを追記する

**注意**: 外部エージェント（delta 等）はローカルにインストールされないため登録対象外。
