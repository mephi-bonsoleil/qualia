    - `~/.ssh/config` にデフォルト鍵を設定

## 決定事項・学び

- hetemlの環境でLAMP環境でサービスを構築する際はPHPで実装する必要がある
- hetemlでは、crontabコマンドではなく、コントロールパネルからcron設定を行う必要がある
- Webアクセス可能な場所にバッチ処理を置く場合、セキュリティのためトークン認証を実装する
- サービス提供用のディレクトリ `services/` を `.gitignore` に追加し、独立したWebサービスを提供できる構造にした
