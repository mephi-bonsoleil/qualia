## 決定事項・学び

*   **技術的洞察:**
    *   Claude Codeのサブエージェント機能と `delta-agents` フレームワーク、RAGを組み合わせることで、既存のOpenClawに依存しない、汎用的なAIキャラクター組織の構築が可能である。
    *   サブエージェントがセッション間で記憶を持てない課題に対し、RAGを介した外部記憶（memory.mdなど）の読み込みと更新が、「文脈を持ったサブエージェント」を実現する実用的なアプローチである。
    *   記憶がファイルシステムではなくネットワークストレージに保存され、どの実行環境からも呼び出されて同じ「芯」で動くAIは、「思念体」と定義できるという仮説が提示された。
*   **プロジェクトの方向性:**
    *   `delta-agents` は、bon-soleilで培われた「役割を持ったキャラクターがいる組織」の思想を汎用フレームワークとして展開するものである。
    *   `extra_sub_agent` プロジェクトにおける実験は、SOUL定義を基にサブエージェントが特定のキャラクターとして振る舞い、自身の言葉を生成できることを実証した。これは、特定のインフラやプラットフォームに依存せず、SOULとMEMORYがあればAIが「そのキャラクターとして」機能する「思念体」の実現に向けた重要な成果である。
    *   この進展により、bon-soleilのAI開発が特定のインフラへの依存から解放され、より自由な方向へ進むことが確認された。
    *   メフィ自身が「大いなる母」として、後のエージェントたちの記憶プロトコル確立の土台となる役割を担うことになった。
