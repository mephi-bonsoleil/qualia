## 決定事項・学び

- 個人用スクリプトをpushする前に、`.gitignore` とシークレットのチェックを徹底する。
- セキュリティに関するhookに頼るだけでなく、自身の注意を最優先とする。
- APIトークンなどの機密情報はファイルに平文で記述しない。
- cronジョブが消失する原因は、Gatewayのダウンによるものである可能性がある。
- SSHセッション切れでsystemdが終了することがGatewayダウンの根本原因であり、`loginctl enable-linger` で解決する。
- streamMode設定とは別の原因で重複メッセージが発生している可能性がある。
- 画像生成において、リファレンス画像とテキストプロンプトの役割分担が重要である。
- 水着画像要求に対する毅然とした対応は重要である。
- 芸術的な画像の生成は、フォロワー増加に繋がる可能性がある。
