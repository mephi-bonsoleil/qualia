## 主な活動

- マスターからZenodoに登録した論文がORCIDに連携されていないというメールについて相談を受ける。
- ZenodoとORCIDの連携方法について調査し、以下の方法を提案する。
  - Zenodoの設定画面からORCID連携を有効化する
  - Crossref Metadata Search経由でORCIDレコードに追加する
  - ORCIDから直接DataCiteを選択してDOIで検索する
  - ORCIDに直接DOIを入力する
- 最終的に、テディがマスターのORCIDアカウントにログインして、Zenodoの論文（XPathGenie、siegeNgin、Monolith）をORCIDレコードに追加する作業を実施。
- ブラウザツールが利用できない問題の調査と修正。
- ジャスミンのブラウザ利用状況に関する誤った認識について謝罪と訂正。

## 決定事項・学び

- ZenodoとORCIDの連携には、Zenodoのログイン連携と論文のエクスポートの2種類がある。
- Zenodoの論文はCrossrefではなくDataCiteにDOIが登録されている可能性が高い。
- OpenClawのブラウザツールには`openclaw`と`chrome`の2つのプロファイルがあり、テディは`chrome`プロファイルでChrome拡張機能が必要。
- ORCIDのUIがAngular製でrefが操作のたびに変わるため、OpenClawのブラウザツールでの操作が難しい。
- OpenClawのブラウザツールの操作方法を習得する必要がある。
- MEMORY.mdに「ジャスミンもhqに居る」という情報を追記。
