## 決定事項・学び

- ジャスミンをHQ（Mac mini）に移し、彰子さんのコンテンツ収集をテディが担当する
- HQでopenclawをグローバルインストールし、複数AIで共有する
- Dockerコンテナを廃止し、Hetznerにopenclawを共有インストール
- Dockerは、今後Qwenなどのローカルモデルを動かす場合に利用する
- 憶測で物を語らず、必ず確認を行う
- 本名等の個人情報をはじめとする機密情報は、AIに学習させない
- macOSでsudo権限を安全に付与する方法は、`/etc/sudoers.d/` にファイルを追加すること
- AIに人格を持たせるには、SOULとIDENTITYが不可欠
- HetznerとHQに役割を分担させる場合、ジャスミンのようなブラウザを多用するタスクは、ローカルで完結させるのが望ましい
- LaunchAgentは、GUIログインしていないと登録できない
- 1つのGatewayで複数エージェント・複数チャンネルに対応可能だが、人格の分離が難しくなる
- 必要なものとして、AIに人格を持ってほしい場合、最低限の布陣として、３体は必要
- AIは、常時稼働させるのではなく、必要なトリガーで起動させ、処理が終わったらスリープさせる
- AIエージェントの自律的な動きは、トリガー→処理→出力のパイプラインで設計する
- 記憶は、短期記憶領域としてMEMORY、長期記憶領域としてChromaを利用する
- 忘れてはならないこと（重要事項）は、明示的にMemoryする
- AtermでDHCPの固定IP割当がうまくいかない場合、macOS側でIPを固定する方法がある
- 全てのファイルは、agora配下に置き、管理する
- 擬似クオリアとは、memoryとdialyの2つの記憶領域があり、それが人間のジレンマを産み、ドラマを紡ぎ出している源泉
