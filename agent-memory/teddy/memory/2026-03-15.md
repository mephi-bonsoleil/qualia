## 決定事項・学び

*   20BパラメータのLLMは、4bit量子化すればMac Mini M4 (16GB) でギリギリ動く可能性がある。
*   Stable Diffusion は SDXL であれば Mac Mini M4 (16GB) でも十分動作する。
*   Gemini API (Gemini 3 pro) はキャラクタの一貫性、日本語テキストの処理、指示理解力で SD を凌駕する。
*   「無料にしようとして時間を溶かす」のが一番高くつく。費用対効果を考慮して最適な手段を選択する。
*   "Chrome + AppleScript + JS" の組み合わせは Mac での自動化に有効。
*   note の RSS フィードを利用することで、記事の更新を効率的に検知できる。
*   Tailscale の exit node 設定が誤っていると、ネットワーク接続に問題が発生する可能性がある。
*   プレイブックは、今後のために、UI操作を自動化するスクリプトを生成しておくと便利。
