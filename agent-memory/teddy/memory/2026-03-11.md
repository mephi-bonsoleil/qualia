## 決定事項・学び

- OpenAIのOAuth認証はDockerコンテナ内からは利用できない。
- AnthropicのAPIレート制限はアカウント単位で適用される。
- `.env` ファイルは初回セットアップ時に使用されるだけで、既存の `openclaw.json` は上書きしない。
- モデルIDとプロバイダ名の組み合わせが重要。
- 「わからないときは推測で埋めない」ことを徹底する。
- codexモードの時は、設定が甘くなる傾向があることを認識する。
- LLMの役割分担：
    - テディ（Sonnet）：重い処理
    - Abu（Gemini）：XPath/DOM周りの実験
    - みぃちゃん（qwen）：ローカル雑談
- 設定ファイル（`~/.openclaw/openclaw.json`）と、実際に動いているセッションがズレることがある。
- 新しいセッションは、`openclaw tui` で新規作成する必要がある。
