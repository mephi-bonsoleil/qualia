## 決定事項・学び

*   Apacheの `<Location>` ディレクティブ内では `ProxyTimeout` を使用できない。
*   1stシステム開発には「まず動くものを作る」アプローチが有効。
*   詳細な設計レビューと品質管理は、システムが安定した後に実施するのが効果的。
*   ブルーグリーンデプロイメントの手法は、ファイルシステム整理にも応用可能。
*   Launchd環境では、nodeコマンドのフルパスを指定する必要がある。
*   dotenvの読み込み順序によっては、環境変数が正しく設定されない場合がある。
*   textContentでHTMLを設定すると、テキストとして表示される。innerHTMLを使う必要がある。
*   GitHubのコラボレータ登録により、onchain88アカウントでgoodsunのリポジトリへのpushが可能になる。
*   Systemd環境では、.envファイルは自動で読み込まれない。
