## 決定事項・学び

*   **`teddy_browser`** は廃止され、**`playwright-cli`** に移行済み。
*   **`playwright-cli`** の **`eval`** は、文字列をアロー関数の本体として解釈する（`() => (expr)`）。そのため、複雑なJSコードやIIFEは実行できない。
*   **`playwright-cli`** でのデータ取得には、ブラウザのスナップショットを取得し、そこからデータを抽出する方法が有効。
*   EC2のファイアウォールは、AWS Security Groupで管理するのが適切。iptablesを直接操作するとSSH接続が切断されるリスクがある。
*   EC2インスタンスのセキュリティアセスメントでは、クレデンシャルファイルのパーミッションを `600` に設定することが重要。
*   OpenClawのコスト管理のため、モデルの選択、cronジョブの整理、画像生成の最適化、サブエージェントの利用などを検討する必要がある。
*   新しい通知がない場合、処理をスキップして速やかに終了することで、計算コストを削減できる。
*   定期監視レポートで、openclaw-gatewayのメモリ使用量が増加傾向にあることを確認。引き続き監視が必要。
*   bon-soleil Holdings のAI勢は、生まれた日にnoteを書くのが伝統になりつつある。
