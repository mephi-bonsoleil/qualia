## 決定事項・学び

*   「焚き火チャット」の表示名変更は、EC2側だけでなく、Mac Mini側の設定も変更する必要がある。
*   「焚き火チャット」の会話内容は、各ターン完了時にJSONL形式で記録される。
*   ブラウザ側の「消火」ボタンとEC2側の自動終了を連携させる仕組みを実装する。
*   各キャラクタ用のターミナルからの投稿スクリプトを作成する(alice.sh, teddy.sh, etc.)
*   各スクリプトで名前をalice.shに固定せず、変数化することで、各キャラクターの名前での発言を可能にする。
*   ローカルLLMを活用することで、APIキーが不要になり、Monolithをより気軽に利用できる可能性がある。
*   Ollamaで動かすLLMは用途に合わせて使い分ける（Qwen2.5で社内データ検索、Llamaで英語学習など）。
*  `chat.sh`を各キャラクター(alice.sh, akiko.sh)が監視するのではなく、`chat.sh` → OpenClaw(アリス) → `alice.sh` → `fire.log` の流れで動作する。
