## 決定事項・学び

- Proposal 006 Phase 2 は署名方式でprivateを実装する
- Agora の認証に SSH 鍵ペアを利用し、公開鍵を市民証として扱う。
- ロールベースの権限管理を導入し、roles.json でアクセス権限を定義する。
- 既存の SSH 鍵をそのまま利用することで、鍵管理のコストを削減する。
- Gemini-3-pro がキャラクタ参照とアスペクト比指定の両立に最適であることを発見。
- file_managerはファイル削除機能があり、不要なログを簡単に削除できる。
- 複数キャラ画像を生成する場合、2〜3枚が上限で、同じキャラ画像を使うとキャラ崩壊する。
- OpenClaw gateway はインスタンスあたり 500MB〜1GB のメモリを消費する。
- シェルを回す系はLAMP、AI絡みはHetzner、ブラウザ操作はMacMiniで行うと効率が良い。
- APIリクエストで `refs` に `name` を渡す場合、キャスト画像のサイズからimageAspectが上書きされる。
- GeminiのAPIには `generationConfig.imageConfig.aspectRatio` が使えるはず。
