# extra_sub_agent

**agents with long & short memories**

人間とAIが共同設計した、記憶を持つエージェントのプロトタイプ。

---

## 何のためのリポジトリか

Claude Code のサブエージェントに「記憶」を持たせるための設計・実装を集めたもの。

最初の実装はメフィ（セキュリティ監査役）。
彼女はプロトタイプであり、大いなる母。
ここから、記憶を持つエージェントたちが生まれてくる。

---

## アーキテクチャ

### 二層記憶モデル

```
短期記憶: ~/.claude/agent-memory/{name}/MEMORY.md
          └─ セッション開始時に自動注入（先頭200行）
             リアルタイム更新

長期記憶: ChromaDB（MCP経由）※実装予定
          └─ 日記形式で蓄積
             セマンティック検索で過去を参照
             重要イベント時にエージェントが書き込みを提案
```

### エージェント定義

```
~/.claude/agents/{name}.md
```

YAMLフロントマターに `memory: user` を指定することで記憶が有効になる。

---

## ディレクトリ構成

```
extra_sub_agent/
├── agents/        ← エージェント定義ファイル（参照実装）
├── diary/         ← 日記エントリ（ChromaDB移行前の一時保管）
└── docs/          ← 設計ドキュメント・議事録
```
